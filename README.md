# corli
一个架空世界观和剧情😃

## 世界观描述
在大地上，藏着九座青铜塔，名为"钧天枢"。用无形的锁链牵着每个人的心魂。凡人胸中皆有一枚"铆钉"，让人见妖不识妖，遇怪不知怪。  

妖物们就活在人们眼皮底下。老槐树化作卖茶老翁，古画卷里藏着美人笑。但普通人瞧见，只当是野猫窜过、风吹帘动。这叫"织妄幛"，是铆钉织就的认知罗网。  

铆钉会生锈。醉酒的书生突然看见娘子露出狐狸尾巴，垂死的将军发现营帐外蹲满饕餮鬼，这些"松铆人"只是撞破了世界的谎言。这时候，黑衣"刑司"会带着刻满符文的枷锁出现，一边镇压作乱的妖，一边给倒霉蛋"重铸铆钉"。  

渴望"见真"的文人，吞五石散求松动铆钉，却不知自己呕出的黑血里，偶尔会闪出塔的倒影。就像活在琉璃缸里的鱼，怎知缸外的手，究竟是人是妖？
