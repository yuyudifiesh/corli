# 唤醒

混沌与虚无之中，祂见你漂浮在星辰之间，祂为你唱响挽歌，祂将你唤醒。

## 剧情预告

::: details 预告 1
淡黄色的天空笼罩着昏黑发暗的土地，斗争、反抗、分裂、冲突，各种各样，空气中充斥着火药的味道。灰质降下的时候，良善成为一种不可获求的期望。

活在金字塔底层就不必抬头看向太阳，活在社会底层就不必仰望星空，活在权贵脚下就不必保守尊严。

土地吃人的时候从不挑食。

END
:::

::: details 预告 2
不要问篝火该不该燃烧，先问寒冷黑暗在不在；  
不要问子弹该不该上膛，先问压迫剥削在不在；  
不要问正义该不该祭奠，先问人间不平还在不在。

END
:::

::: details 预告 3
Nor will it at all detract from him, dramatically regarded, if either by birth or other circumstances, he have what seems a half wilful overruling morbidness at the bottom of his nature. For all men tragically great are made so through a certain morbidness. All mortal greatness is but disease.

END
:::
