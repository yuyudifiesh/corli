# HU-5 跨过旧夜

## 预告
清晨的光又一次漫过慈利平原，穿过山体间缥缈的薄雾，落在安置点那片 45 岁的建筑群上。外墙的锈色在光线里一层层晕开，像老照片上褪不干净的底色。我站在阳台上，忽然想起爷爷说过的那些话。

那是华川建国初期的某个傍晚，爷爷坐在胜利区平原边缘的石墩上，我趴在他膝盖边，看远处的能量桩把天空染成暖黄色。

“爷爷，太阳为什么是热的？”

“因为它是咱老百姓自己托起来的。”

我不懂。

他指着东边，那里有新建的厂房、刚立起的能量桩、远处工地上的脚手架，还有一群群收工回来的人，肩上扛着工具，嘴里哼着调子。

“你看那些人，”他说，“每一个心里都有一团火。有人烧砖，有人炼钢，有人教书，有人种灰质稻。火不大，但人多。亿万团火凑在一起，就能把大太阳从东边抬起来。”

“那月亮呢？月亮为什么冷？”

“月亮不会发光，它只会借。借别人的光，照自己的脸。但太阳不一样，太阳是自己烧出来的，是千千万万双手托出来的，是实打实干出来的。”

他又指向远处。那里有刚建成的第一座灰质结晶体发电站，有第一批从技校毕业的年轻工人，有墙上刷着的标语。

<p><strong>自力更生，艰苦奋斗</strong></p>

“记住，咱华川人不靠借光过日子。咱们自己就是光源。”

后来先驱者们都走了。一个接一个，像秋天里的灰质稻，熟了，倒了，归了土。

那段时间，安置点的夜似乎特别长。有人坐在楼顶上发呆，有人对着北方唱老歌，有人把先驱者的画像挂在墙上，看了又看。

但日子还得过。

于是，厂房继续转，学校继续上课，能量桩继续供热。年轻的一代接过扳手、教鞭和药箱，继续往前走了。他们改了工艺，修了章程，添了新机器，也添了新毛病。有些东西丢了，有些东西变了，有些东西被压在箱底，落满了灰。

可有一件事没变。

每到清晨，阳光还是会穿过薄雾，打在那些老建筑的墙上。锈色的光，暗淡的，却又是实的。

那是自己烧出来的光，不是借的。

我站了很久，直到楼下传来早餐铺的吆喝声。

太阳照常升起。
